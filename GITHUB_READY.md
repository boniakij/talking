## 🚀 Upload Methods (Choose One)

### Method 1: Automated Script (Easiest)

Just double-click: **`upload-to-github.bat`**

The script will:
1. Check if Git is installed
2. Configure Git (if needed)
3. Initialize repository
4. Add all files
5. Create commit
6. Add remote repository
7. Push to GitHub

### Method 2: Manual Commands

```bash
# 1. Initialize Git
git init

# 2. Add all files
git add .

# 3. Commit
git commit -m "Initial commit: Complete BaniTalk platform"

# 4. Add remote (replace with your URL)
git remote add origin https://github.com/yourusername/banitalk.git

# 5. Push to GitHub
git branch -M main
git push -u origin main
```

### Method 3: GitHub Desktop

1. Download GitHub Desktop: https://desktop.github.com/
2. Open GitHub Desktop
3. File → Add Local Repository
4. Select your project folder
5. Click "Publish repository"

## 🔐 Authentication

You'll need a **Personal Access Token** (not your password):

1. Go to: https://github.com/settings/tokens
2. Click: "Generate new token (classic)"
3. Name: "BaniTalk Upload"
4. Select: `repo` (full control)
5. Click: "Generate token"
6. **Copy the token** (save it somewhere safe!)
7. Use this token as password when pushing

## 🐛 Troubleshooting

### "Git not found"
Install Git from: https://git-scm.com/

### "Authentication failed"
Use Personal Access Token, not password

### "Repository not found"
Make sure you created the repository on GitHub first

### "Permission denied"
Check your Personal Access Token has `repo` scope

### "Large files warning"
Some files might be too large. Check `.gitignore` is working.
